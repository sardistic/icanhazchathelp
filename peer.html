If you're experiencing a "peer connection failed" error when attempting to broadcast your camera, this is a known issue with the recent Chrome 127 update. This problem also affects other Chromium-based browsers such as Edge, Opera, and Brave.

sardistic August 31, 2024 12:00 PM

As an alternative, you can try using non-Chromium browsers like Firefox or Safari. If you encounter issues with those browsers as well, you can use the standalone version of Chrome provided below. This version doesn't require you to uninstall your current Chrome or go through any installation process.

sardistic August 31, 2024 12:01 PM

For Mac, download the v126.66 chrome-mac.zip file, extract it, and run the chrome application inside the extracted folder.

For Windows, download the v126.66 chrome-win32.zip file, extract it, and run the chrome.exe application inside the extracted folder.

If you're trying to use the Dark Reader extension, you'll need to download the darkreader-chrome.zip file. Follow these steps to install it:

  • Download the ZIP file from the link above.
  • Extract the ZIP file to a folder on your computer.
  • Open Chrome and go to chrome://extensions/.
  • Enable "Developer mode" at the top right corner.
  • Click on "Load unpacked" and select the folder where you extracted the ZIP file.
  • The Dark Reader extension should now be installed and active.
